# Archie — Public Documentation

Product and operations documentation for Archie, a self-hosted VPN management and
security operations platform. These documents are written for operators,
customers, collaborators, and prospective adopters.

| Document | Purpose |
|----------|---------|
| [ARCHIE_OVERVIEW.md](ARCHIE_OVERVIEW.md) | What Archie is, who it is for, and what makes it different |
| [INSTALL_AND_OPERATIONS.md](INSTALL_AND_OPERATIONS.md) | Installing and operating Archie on your own host |
| [SECURITY_MODEL.md](SECURITY_MODEL.md) | Authentication, enforcement, and security operations model |

These documents are intentionally free of environment-specific details. Where a
value depends on your deployment, a placeholder such as `your-domain.example` is
used.
